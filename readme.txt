cutewiki_symbian folder has all the source files related to cutewiki wikipedia reader application.

cutewiki presented here is of version 2.0 and is newer version available in ovi store (which is 1.3).

The delta between 1.3 and 2.0 are:

1. russian language is added.
2. new qt-components are used to show errors.
3. half qwerty keypad used.
4. layout fixes for webkit usage.

cutewiki (all the source file and graphics used) is release under GPL license.
So use it accodingly.
